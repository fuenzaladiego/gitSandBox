Puedes trabajar en distintas ramas al mismo tiempo
Puedes fusionar las ramas en una si asi lo deseas

para crear ramas
git branch <nombre rama>

para ver las ramas del projecto usamos simplemente
git branch

para cambiar de ramas 
git checkout cambio rama

para fucionar cosas de ramas debemos
1) Ir a la rama main
2) git merge <nombre rama a fusionar>
